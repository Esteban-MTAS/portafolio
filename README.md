# Portafolio Web - Esteban Josue Mata Acosta

Este proyecto es una página web personal desarrollada para presentar mi perfil profesional, habilidades técnicas, proyectos realizados, currículum y medios de contacto.

## Objetivo del proyecto

El objetivo principal de este portafolio es contar con una plataforma personal donde pueda mostrar de forma organizada mi información profesional, mis proyectos de desarrollo y mis canales de contacto.

## Secciones principales

El sitio está dividido en las siguientes secciones:

- Inicio
- Sobre mí
- Habilidades
- Proyectos
- CV
- Contacto

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome
- GitHub Pages
